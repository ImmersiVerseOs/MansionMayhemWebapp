What Happens Here

  • Sticky Header Shows scenario number, type badge (CONFRONTATION), title, and countdown timer
  • Dramatic Narrative Multi-paragraph story sets the scene - written like a reality TV moment
  • Multiple Choice Response User selects ONE option (radio button behavior)
  • Three Response Strategies Each option has a title and description explaining the consequences
  • Optional Voice Note Users can record a 30-60 second audio reaction (premium feature)
  • Submit Button Sticky footer button - disabled until user selects an option
  • After Submission Shows confirmation modal, then returns to dashboard
  • What Happens Next User waits for other cast to respond. In 4-24 hours, scenario closes and outcomes are revealed

Complete Game Loop

After submitting, users return to the dashboard and wait for the next scenario. The game continues with 1-3 scenarios per day for 3-7 days until elimination or finale.
