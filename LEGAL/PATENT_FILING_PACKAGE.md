- **Option A:** Create professional diagrams using Draw.io/PowerPoint
- **Option B:** Use ASCII diagrams from PATENT_DIAGRAMS files (USPTO accepts)
- Save as PDF at 300 DPI

**Step 4: File Online**
1. Log into USPTO.gov
2. Go to "EFS-Web" (Electronic Filing System)
3. Select "Provisional Patent Application"
4. Upload documents:
   - Application PDF
   - Diagrams PDF
   - Cover sheet (auto-generated)
5. Pay filing fee via credit card
6. Submit

**Step 5: Receive Confirmation**
- USPTO sends filing receipt within 1-2 weeks
- Receipt includes application number
- **SAVE THIS:** You need it to file full patent within 12 months

**Step 6: Repeat for Second Patent**
